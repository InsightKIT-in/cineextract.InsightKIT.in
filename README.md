# cineextract.InsightKIT.in

> CineExtract — an AI-powered movie and TV show extractor built by InsightKIT.

## Overview

cineextract.InsightKIT.in transforms a folder full of screenshots into a clean list of movies and TV shows within seconds.

Simply drop all your screenshots at once, and the AI scans every image, identifies titles, removes duplicates automatically, and generates an organized collection that you can edit or export.

## Features

- 🎬 Extract movie and TV show names from screenshots
- 📸 Batch upload multiple images at once
- 🤖 AI-powered recognition using Groq Vision API (LLaMA 4)
- ✅ Automatic duplicate removal
- ✏️ Manually edit extracted results
- 📄 Export data as CSV, TXT, or JSON
- ⚡ Fast processing with no manual typing
- 💻 Runs as a single HTML file
- 🔒 No backend, no login, and no installation required

## How It Works

1. Drop one or multiple screenshots into the application
2. AI analyzes every image and detects movie or TV show titles
3. Duplicate entries are automatically removed
4. Review or edit the extracted list if needed
5. Export the final collection in your preferred format

## Why It Exists

People often save screenshots of recommendations from social media, chats, or streaming apps but never organize them.

CineExtract converts those scattered screenshots into a structured watchlist automatically, saving time and eliminating manual work.

## Tech Concept

- Batch image processing
- Vision AI-powered text and title extraction
- Automatic deduplication
